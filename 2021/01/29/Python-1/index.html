整数类型

int 无限长度

m //n 整数除法，获得商的整数部分

divmod(m, n) 分别得到m / n和m % n

m ** n 乘方，求m的n次方

二进制 0b

八进制 0o

十六进制 0x

进制转换

十六进制 hex()

float 17位有效数字

进制转换导致精度误差，在判断是否相等时可以使其小于一个极小的数值

a+bj

实部 (a+bj).real

import cmath 适用于复数

cmath.polar(a + bj) 把直角坐标转换为极坐标

cmath.rect(a, b) 把极坐标转换为直角坐标

\n 换行

len(s)

s[start:end:step] 切片(slice),左闭右开,向后step位取字符

s * n n个s相加

'x' in s 判断s中是否有子串x

s.strip() 删除s前后所有空格

s.lstrip()/rstrip() 删除s前/后部所有空格

s.isalpha()/isdigit() 判断s是否全部字母/数字

s.isalnum() 判断s是否仅包含字母和数字

s.split(' ') 以’ ‘分割s

'-'.join([s1, s2, s3]) 以’-‘连接s1,s2,s3

s.upper()/lower()/swapcase() 大小写转化

s.replace('a', 'b') 替换子串a为子串b
