列表和元组

元组是不可更改的

s = []或s = list() 创建列表s

s = ()或s = tuple() 创建元组s

s.append(obj) 在s末端插入

s.insert(index, obj) 在index位置插入

s.extend(obj) 在s末尾连接

s.pop() 删除s末尾元素

s.pop(i) 删除地址为i的元素

s.remove(obj) 删除内容为obj的元素

s.clear() 将列表清空

s.reverse() 按照原来相反顺序排列

s.sort() 从小到大

s.sort(reverse = true) 从大到小

s.index(obj) obj首次出现的位置

s.count(obj) obj出现的次数

s.remove(obj) 将obj的首次出现删除

字典

dic = {}或dic = dict() 创建字典

字典中保存标签 : 数据值(key : value)

dic = dict.fromkeys() 批量添加数据项

dic.update(obj) 更新dic

dic.update(key = value) 批量添加数据

del dic[obj] 删除标签为obj的数据

dic.pop(obj) 删除并返回

dic.popitem() 随机删除并返回

dic.clear() 清空字典

dic.len() 数据项对数

dic.keys() 返回全部标签

dic.values() 返回全部数据值

dic.items 将全部数据项以二元元组形式返回

obj in dic 判断obj是否存在

集合

集合是可变的，只有keys

集合自动忽略重复数据，只能加入不可变类型数据

aset = {} 创建非空集合

aset = set() 创建集合

aset.add(obj) 添加数据

aset.update(obj) 批量添加数据

aset.remove(obj)/discard(obj) 删除

aset.pop() 随即删除并返回

aset.clear() 清空

aset.len() 包含元素数目

obj in aset 判断是否存在

aset.isdisjoint(obj) 判断两个集合是否有交集
